# SQL Injection Lab

## Objective
Understand SQL Injection attacks in a controlled environment.

## Tools
- DVWA
- Firefox
- Metasploitable2
- Kali Linux
- sqlmap

## Skills Demonstrated
- Web Enumeration
- SQL Injection Testing

## Impact 
- Data stealed from sql injection
- Unauthorized users can access sensitive information

## Mitigation technique
- Prepared statement
- Input validation and sanitization
- Restricting database permission
- Using web applicatin firewall
